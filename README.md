# Automated-WhatsApp-Message-Sender-using-Python
This project demonstrates how to send an automated WhatsApp message using Python with the help of the pywhatkit library. It allows users to schedule and send messages to a specific WhatsApp number at a given time.

About pywhatkit
pywhatkit is a popular Python library that provides automation features such as:
Sending WhatsApp messages
Playing YouTube videos
Performing Google searches
Converting text to handwriting
Basic automation tasks


For WhatsApp messaging, it works by:
Opening WhatsApp Web in your browser
Automatically typing and sending the message at the scheduled time

Technologies Used:
Python
pywhatkit
WhatsApp Web
